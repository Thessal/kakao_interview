## How to run

```
# Set up python 3.8.6
brew install pyenv 
pyenv install 3.8.6
pyenv init
eval "$(pyenv init -)"
pyenv shell 3.8.6

# Dependencies
poetry env use 3.8.6
poetry install

# Run
poetry run python demo.py
```
